### 题目描述
给定一个字符串 s，找到 s 中最长的回文子串。你可以假设 s 的最大长度为 1000。

示例 1：

输入: "babad"
输出: "bab"
注意: "aba" 也是一个有效答案。

示例 2：

输入: "cbbd"
输出: "bb"

### 解题
#### 暴力法
会超出时间限制
```
public String longestPalindrome(String s) {
        if (s.length() == 1) {
            return s;
        }
        int max = 0;
        String result = "";
        for (int i = 0; i < s.length();i++) {
            for (int j = i; j < s.length();j++) {
                String temp = s.substring(i,j+1);
                if (isHuiwen(temp) && temp.length() > max) {
                    max = temp.length();
                    result = temp;
                }
            }
        }

        return result;
    }

    public boolean isHuiwen(String s) {
        int start = 0;
        int end = s.length()-1;

        while (start <= end) {
            if (s.charAt(start++) == s.charAt(end--)) {
                continue;
            } else {
                return false;
            }
        }
        return true;
    }
    
    改进：减少循环次数，如果当前子串的长度小于max，可以直接跳出循环

#### 中心扩散法
```
    public String longestPalindrome(String s) {
        if (s.length() == 1) {
            return s;
        }
        int max = 0;
        String result = "";
        for (int i = 0; i < s.length()-1;i++) {
            String len1 = aroundCenter(s, i,i);
            String len2 = aroundCenter(s, i, i+1);

            if (len1.length() > len2.length() && len1.length() > max) {
                max = len1.length();
                result = len1;
            }

            if (len2.length() > len1.length() && len2.length() > max) {
                max = len2.length();
                result = len2;
            }
        }

        return result;
    }

    public String aroundCenter(String s, int left, int right) {
        String len = "";
        while (left >=0 && right<=s.length()-1 && s.charAt(left) == s.charAt(right)) {
            len = s.substring(left, right+1);
            left--;
            right++;
        }
        return len;
    }

    public String longestPalindrome(String s) {
            if (s.length() == 1) {
                return s;
            }
            if(s.length()<0 || s.equals(""))
                return "";
            
    
            int start = 0;
            int end = 0;
            String result = "";
            for (int i = 0; i < s.length();i++) {
                int len1 = aroundCenter(s, i,i);
                int len2 = aroundCenter(s, i, i+1);
    
                int len = Math.max(len1, len2);
                if (len > end - start) {
                    start = i - (len - 1) / 2;
                    end = i + len / 2;
                }
    
            }
    
            return s.substring(start, end+1);
        }
    
        public int aroundCenter(String s, int left, int right) {
            while (left >=0 && right<s.length() && s.charAt(left) == s.charAt(right)) {
                left--;
                right++;
            }
            return right - left - 1;
        }
